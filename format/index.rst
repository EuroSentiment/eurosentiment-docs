The Eurosentiment Format
========================

The Eurosentiment format is an extension of already stablished formats for Linguistic Linked Data (lemon_, NIF_, Onyx_, Marl_, etc.) for their use in Sentiment Analysis.
It covers the description of lexica and corpora on the language resource side, and the results from services.

In the following sections we will cover both cases individually.

.. _lemon: http://lemon-model.net
.. _NIF: http://persistence.uni-leipzig.org/nlp2rdf/
.. _Marl: http://www.gsi.dit.upm.es/ontologies/marl
.. _Onyx: http://www.gsi.dit.upm.es/ontologies/onyx
